### GOLANG 

Local setup for GOPATH , GOBIN etc:



Add this to ~/.bash_profile or ~/.bashrc

```

# Add go to path:
PATH=${PATH}:/usr/local/go/bin

```

Now you should have the actual go binary in your path and be able to run:

```

go run

go version

go install

go env

```
